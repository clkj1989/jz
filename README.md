# Jz JavaScript Popup

> 一个轻量、简单的弹窗插件

----------
### 功能（Function）

- `jz.msg` （ 提示消息，仅仅是内容 ）

- `jz.alert` （ 提示消息，类似系统提示 alert ）

- `jz.confirm` （ 询问提示，类似系统提示 confirm ）

- `jz.iframe` （ 弹出 iframe ）

- `jz.tip` （ 气泡提示 ）
